About ITCONNECT

One platform for smarter event registration and user management.

ITCONNECT is designed to streamline registration for multiple user roles, helping institutions and communities onboard admins, organizers, and students with custom profile fields.

What we offer

This project combines a custom Django user model with role-specific profiles so registration can adapt based on the user's role. It keeps common account fields consistent while allowing each type to capture only the data that matters to them.

Custom user profiles

Each account type gets a dedicated profile model so admins, organizers, and students store only relevant information.

Reusable layout

All pages now extend a shared base template so navigation, header, and footer are consistent across the app.

Expandable structure

You can add more pages and features without copying boilerplate HTML, since each page extends the base template.
